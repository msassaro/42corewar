Synopsis:

./asm   [ -bcdehiknrs (name if -n)]   [(only 1) file]

Flags: 

-b         display binary output.

-c         display compilation caracteristics.

-d         debug mode (step by step).

-e         display detailed errors.

-h         user shell manual (or --).

-i         information mode.

-k         display encoding details.

-n         define a new name (./asm -n "new name" champ.s)

-r          file requirements

-s          This is style !!! YEAH !!!

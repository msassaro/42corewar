Synopsis:

./asm   [ -bcdehiknrs (name if -n)]   [(only 1) file]

Description: 

The ASM compiler is as its name indicates: a compiler. In the Corewar project, it will take as input a champion written in corewar asm language and will convert it into bytecode.

You can receive one input only. This input will be the champion. The file extension need to be of type ' .s ' to be acceptet such as "champ.s"

Then the ASM will create an outfile with ' .cor ' extension as "champ.cor".

champ.s    ------->   [ASM]    ------->   champ.cor

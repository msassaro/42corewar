Description:

The COREWAR project consist of a battle of programs into a virtual arena. It's likely to create a COMPILER and a PROCESSOR (virtual machine). And of curse a program to fight into this virtual arena which could be compared to a VIRUS.

The Virus: champion

You need to have champions to fight into the corewar virtual arena. Those champions are written into a special programming language: The Corewar asm language.

There are 16 instructions allowed. This asm language will be compiled for the virtual machine to interpret it. This compilation process will be the job of the ASM Compiler.

The Compiler: ASM

The ASM is the compiler. It will "translate" the corewar asm language into binar that could be interpreted by the virtual machine.
It will read the champion file written in asm language then parse it, check for the eventual issues, encode the parameters and create a new file written under bytecode.

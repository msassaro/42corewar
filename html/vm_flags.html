Synopsis:

./corewar [ -dnsv dm ] [ value ] [champ1.cor ...]

Flags: 

-d         dump the memory at X cycles

[-d 12] : dump the memory 12 cycles

-n         precision of each player number

When -n option is activated, all players must have nb precision
[-n 5 player.cor] : give the nb 5 to player.cor.

-s         pause the process at each X cycle interval

-v         graphical ncurses mode

-dm        dump the memory at each X cycles

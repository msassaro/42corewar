The Corewar Virtual Machine is like a Processor. We have registers and a memory table
in which we will execute the code and the instructions itself.
The virtual machine correspond to an infinite loop.
First we will check the input received and store them. We will then have the
champions name, instruction, player number and so on ...
After that we will have an infinite loop.
We will execute each process depending on the Cycle and the instruction
